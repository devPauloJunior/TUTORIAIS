Bootstrap - framework direcionada para CSS
Trabalha diretamente com a propriedade "class" do HTML


Bootstrap utiliza as estruturas de "breakpoint" que são pontos especificos para deixar o site responsivo.
breakpoints são criados atraves do "media query"
bootstrap trabalha também com o padrão "mobile first" que esta diretamente ligado com a resolução de tela do menor tamanho para o maior!!!.

classes de breakpoint que vamos utilizar:
sm - small ou pequeno
md - middle ou medio
lg - large ou grande


containers - são a entidade mais importante do bootstrap.


Obs: O bootstrap não é 100%, para corrigir esse problema temos que sempre ter em nossa estrutura nosso proprio CSS.

## Nomenclaturas Padrões ##

HTML - index.html
CSS -  main.css

IMAGENS NO HEADER - hd_nome_da_imagem
IMAGENS NO MAIN - mn_nome_da_imagem
IMAGENS NO FOOTER - ft_nome_da_imagem

m - margin
p - padding

t - top
b - bottom
s(start) - left 
e(end) - right

0 - elimina o elemento
1 - 0.25
2 - 0.50
3 - Normal
4 - 1.50
5 - 3.0
auto

x - eixo horizontal
y - eixo vertical

BORDER
Obs: bordas sempre devem ser aplicadas com duas classes principais
